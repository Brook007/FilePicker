# FilePicker
Android文件选择器

[![](https://jitpack.io/v/Brook007/FilePicker.svg)](https://jitpack.io/#Brook007/FilePicker)

## 引入依赖

1. 根项目的build.gradle中加入以下代码

```gradle
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
```

2. 在需要的模块加入以下的依赖

```gradle
dependencies {
	implementation 'com.github.Brook007:FilePicker:1.0.0-beta'
}
```
